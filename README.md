# 🏥 Hospital Readmission Risk Analysis

## 📌 Project Overview
This project focuses on analyzing healthcare data to identify key factors contributing to 30-day hospital readmissions. Reducing readmissions is critical for improving patient outcomes and lowering healthcare costs.

## 🎯 Objectives
- Identify patterns and trends in patient readmissions  
- Analyze demographic and clinical factors affecting readmission rates  
- Provide data-driven insights for healthcare decision-making  

---

## 📊 Dataset
This project uses a simulated healthcare dataset containing:
- Patient demographics (age, gender)
- Admission details (length of stay, diagnosis)
- Medical history (chronic conditions, prior admissions)
- Readmission status (within 30 days)

---

## 🛠️ Tools & Technologies
- Excel / CSV for dataset handling  
- Python (Pandas, NumPy) *(optional)*  
- Tableau / Power BI for visualization  

---

## 🔍 Key Analysis Performed
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Trend Analysis  
- Correlation Analysis  

---

## 📈 Key Insights
- Patients with chronic conditions showed higher readmission rates  
- Frequent prior admissions strongly correlate with readmission risk  
- Shorter hospital stays may indicate premature discharge  
- Elderly patients had higher likelihood of readmission
